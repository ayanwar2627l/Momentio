STEP 1:- inside backend
npm init -y
This creates:
backend/package.json

Step 2: Install backend packages

npm install express mongoose dotenv cors bcryptjs jsonwebtoken
npm install -D nodemon

express       → creates backend server and API routes
mongoose      → connects Node.js to MongoDB using models/schemas
dotenv        → loads secret values from .env file
cors          → allows frontend to talk to backend (cross origin resource sharing)     
bcryptjs      → hashes passwords
jsonwebtoken  → creates/verifies login tokens
nodemon       → restarts server automatically during development

JWTs are commonly created using jwt.sign(...) and checked using jwt.verify(...)
